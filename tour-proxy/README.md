# AccessMap Tour Proxy

Small HTTPS-deployable proxy for the AccessMap Journey 360 tour links.

Local test:

```sh
npm start
```

Then open:

```text
http://localhost:5199/tour/21673327b1cc0f62
```

Deploy target:

- Render Web Service
- Railway Node service
- Any Node host with HTTPS

Use this command:

```sh
npm start
```
